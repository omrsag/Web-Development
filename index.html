SEO

SEO is the process of improving the web page so that it ranks higher in search engine results (like Google), which increases visibility and attracts more visitors.

Factors can influence SEO:

  • Use a Proper Title Tag ( Keep it under 60 characters ):

    A clear, keyword-rich title helps search engines understand the page and encourages users to click.

  • Meta Description ( Summarizes the page for users ).

    It summarizes the page content, helping users know what to expect before clicking.

  • Alt Attributes for Images.

    They are essential for accessibility (screen readers) and also help search engines understand images, which can improve SEO.

HTML

Basic Structure of an HTML Document:

  • <!DOCTYPE html>:

    Declares the document as HTML5.

  • <html>:

    Root element that wraps all HTML content.

  • <head>:

    Contains meta info (like title, CSS links, charset).

  • <body>:

    Holds the visible content shown on the webpage.

Semantic HTML5 Tags:

    Semantic tags describe the meaning of the content, improving accessibility, SEO, and code clarity.

  • Example:

    <article>
    <h2>Blog Post Title</h2>
    <p>This is a blog post content.</p>
    </article>

  • <article>:

    defines a self-contained piece of content.

  • Benefit:

    Helps search engines and screen readers understand page structure better.

Common Block-level and Inline Elements:

  1. Block-level Elements:

    • <div>: Generic container.

    • Example:

      <div class="container">
      </div>

    • <p>: Paragraph.

    • Example:

      <p>This is a paragraph.</p>

  2. Inline Elements:

    • <span>: Generic inline container.

    • Example:

      <p>this is a <span>highlighted</span> word.</p>

    • <a>: Link to another page.

    • Example:

      <a href="https://example.com">Visit Site</a>

Meta Tags

Meta tags provide metadata about the HTML document and are placed inside the <head>.

  • <meta charset="UTF-8">:

    Defines the character encoding (usually UTF-8), ensuring proper display of text.

  • <meta name="description" content="...">:

    Provides a short summary of the page for search engines and social media.

  • <meta name="keywords" content="...">:

    Historically used to define SEO keywords. Now has less impact but still shows content intent.

  • <meta name="viewport" content="...">:

    Essential for responsive design. Helps control layout on mobile devices.
